Flexbox

Flexbox can be used to format elements in css. It can be accessed by putting display: flex; in a parent elements class.

  • Display flex aligns all the child elements evenly.
  • Use flex-direction to change if the elements align in a row or column.
  • Justify-content aligns the elements on the x-axis (main axis)
  • Align-items aligns the elements on the y-axis (cross axis)
