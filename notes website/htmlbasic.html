HTML Basics

HTML stands for hypertext markup language. Almost every modern webpage uses HTML as well as CSS and Javascript. HTML is made up of tags. Below are the main tags of HTML.

  • HTML, HEAD, TITLE and BODY tags are the four basic tags
  • HEAD is where you place information that the browser needs to create the web page
  • TITLE is the text that you want to appear in the title bar/tab of the browser
  • H1...H6 (largest to smallest)
  • P is for a paragraph of text (a line break will automatically be used at the end of the paragraph)
  • BR is used for a line break (like pressing the Enter key)
  • STRONG (used to make text bold)
  • EM (used to italicize text)
  • BUTTON (can be styled and can add functionality to the web page)
  • UL and LI are used to create unordered (bulleted) lists such as navigation items
  • OL and LI are used to create ordered (numbered) lists
