CSS Basics

CSS stands for cascading style sheets and is often used in combination with html and javascript. It allows developers to format their websites better and adds many options to improve the overall quality of the website. Flexbox and grid can be used in css to help align elements easily on the page. Checkout flexbox under css in the header to learn more.

  • Use the name of a tag followed by brackets to create a CSS class.
  • Put properties of the brackets such as color: and background: to edit the certain properties of a tag.
  • Ex:
    • body{
    background: blue;
    font-family: monospace;
    font-size: 2rem;
    }
  • To create a custom class, put a period in front of the name of the class, then add the attribute class=”class-name” to the element.
    • .blue{
    background-color: #05386B;
    }
  • To create a class for an element by an id, use #
    • #box{
    color: #5CDB95;
    text-align: center;
    font-family: monospace
    font-size: 35px;
    }
